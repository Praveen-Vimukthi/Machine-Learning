## What is Matplotlib? üß©
Matplotlib is a Python library used to make 2D plots (lines, bars, pies, histograms, images, etc.). Think of it as a painter for numbers ‚Äî you give it data and it draws charts.

There are two styles to use it:
- **Pyplot (state-machine)**: quick and simple (`plt.plot(...)`, `plt.show()`).
- **Object-oriented (recommended)**: more control (`fig, ax = plt.subplots(); ax.plot(...)`).

## Quick example: pie chart üç© (with code + explanation)

```python
import matplotlib.pyplot as plt

activities = ['Sleep', 'Work', 'Play', 'Study']
hours      = [8, 8, 4, 4]

fig, ax = plt.subplots(figsize=(6,6))              # create a square figure
ax.pie(
    hours,
    labels=activities,
    autopct='%1.1f%%',
    startangle=90,
    colors=['#ff9999','#66b3ff','#99ff99','#ffcc99'],
    shadow=True
)
ax.set_title("üç© Daily Routine Pie Chart")
plt.show()
```

### What each argument means (plain language)
- `hours` ‚Äî the numbers for each slice (how big each slice is).
- `labels=activities` ‚Äî label text for each slice.
- `autopct='%1.1f%%'` ‚Äî **how the percent text is shown on each slice**:
  - `%1.1f` is a C-style format: show the number as a float with **1 decimal place** (e.g., `12.3`).
  - `%%` is how you include a literal percent sign in that format string, so result looks like `12.3%`.
  - So `'%1.1f%%'` ‚Üí `8.3%`, `50.0%`, etc.
  - You can also use `autopct='%.0f%%'` (no decimals) or pass a **function** to format custom text.
- `startangle=90` ‚Äî rotate the start of the first slice by 90 degrees (so it begins at the top). Angle is in degrees.
- `colors=[...]` ‚Äî list of colors (hex strings, color names, tuples). If you omit it, Matplotlib uses default theme colors.
- `shadow=True` ‚Äî draw a shadow under the pie for a 3D look.

## Fancy: show percent **and** absolute value (custom autopct) üßÆ

```python
def make_autopct(values):
    def my_autopct(pct):
        total = sum(values)
        val = int(round(pct * total / 100.0))
        return f"{pct:.1f}%\n({val} hrs)"
    return my_autopct

ax.pie(hours, labels=activities, autopct=make_autopct(hours), startangle=90)
```

This prints like:

```
20.0%
(8 hrs)
```

## More pie options (mini cheat-sheet) üìã

| Parameter | Purpose | Example / Notes |
|---|---:|---|
| `explode` | separate slices a bit (tuple of floats) | `explode=(0,0.1,0,0)` moves second slice out |
| `wedgeprops` | style slice boundaries (dict) | `wedgeprops={'edgecolor':'w','linewidth':1}` |
| `textprops` | style the label text (dict) | `textprops={'fontsize':12}` |
| `pctdistance` | radial position of percent text (0..1+) | smaller ‚Üí closer to center |
| `labeldistance` | radial position of labels | >1 places labels outside the pie |
| `counterclock` | draw slices counterclockwise (True/False) | default True |
| `radius` | size of pie | `radius=1.2` makes it bigger |

## Explain common function arguments üîç

**Plot styling**
- `color='red'` or `color='#ff0000'` ‚Äî set color (name, hex, RGB tuple).
- `linewidth=2` or `lw=2` ‚Äî thickness of lines.
- `linestyle='--'` or `'-'` or `':'` ‚Äî dashed, solid, dotted.
- `marker='o'` or `'^'` ‚Äî point marker shapes (circle, triangle, etc.).
- `markersize=6` ‚Äî size of markers.
- `alpha=0.7` ‚Äî transparency: `0.0` (invisible) to `1.0` (opaque).

**Scatter-specific**
- `s=50` ‚Äî marker size (area).
- `c='blue'` or `c=[list]` ‚Äî color or list of colors for points.
- `cmap='viridis'` ‚Äî color map when `c` is numeric.

**Bar**
- `width=0.8` ‚Äî bar width.
- `align='center'` or `'edge'` ‚Äî bar alignment.

**Histogram**
- `bins=10` ‚Äî number of bins (how many bars).
- `density=True` ‚Äî make histogram show probability density (area sums to 1).

**Figure & axes**
- `figsize=(8,6)` ‚Äî width and height in inches.
- `dpi=100` ‚Äî dots per inch (pixel density).
- `title`, `xlabel`, `ylabel` via `ax.set_title(...)`, `ax.set_xlabel(...)`, etc.
- `legend()` ‚Äî show legend (use `label='name'` in plots to populate).

**Save**
- `plt.savefig('image.png', dpi=300, bbox_inches='tight')` ‚Äî high-res save and trim whitespace.

## Object-oriented style example (recommended) üß≠

```python
import matplotlib.pyplot as plt
fig, ax = plt.subplots(figsize=(7,4), dpi=100)
x = [0,1,2,3,4]
y = [0,1,4,9,16]

ax.plot(x, y, color='tab:blue', marker='o', linestyle='--', linewidth=2, label='squares')
ax.set_title("Line plot example ‚úÖ")
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.grid(True)                 # show grid
ax.legend()                   # show legend
plt.show()
```

## Styling tips & nice tricks ‚ú®
- Use `plt.style.use('seaborn')` or `'ggplot'` for prettier defaults.
- Use `fig, ax = plt.subplots()` and work with `ax` for clearer code and multiple subplots.
- For multiple plots in one figure: `fig, axs = plt.subplots(2, 2, figsize=(10,8))`.
- Create a donut chart by using `wedgeprops={'width':0.4}` in `pie()`.

Donut example:

```python
ax.pie(hours, labels=activities, autopct='%.0f%%', startangle=90,
       wedgeprops={'width':0.4, 'edgecolor':'w'})
```

## Short example: full annotated pie (donut + labels + custom percent) üç©‚ú®

```python
import matplotlib.pyplot as plt

activities = ['Sleep', 'Work', 'Play', 'Study']
hours = [8, 8, 4, 4]

def autopct_generator(values):
    def my_pct(pct):
        total = sum(values)
        val = int(round(pct * total / 100.0))
        return f"{pct:.1f}%\n({val} hrs)"
    return my_pct

fig, ax = plt.subplots(figsize=(6,6))
ax.pie(
    hours,
    labels=activities,
    autopct=autopct_generator(hours),
    startangle=90,
    explode=(0, 0.05, 0, 0),                       # emphasis 2nd slice
    wedgeprops={'width':0.45, 'edgecolor':'white'},# donut effect
    textprops={'fontsize':11},
    pctdistance=0.75,                              # place percent text a bit out
)
ax.set_title("üç© Daily Routine ‚Äî Donut + Values")
plt.show()
```
